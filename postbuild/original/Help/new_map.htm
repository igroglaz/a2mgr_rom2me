Creating new Map

This mode is available by pressing from main tools panel or by selecting New from the main menu or by pressing <Ctrl> + N

When you enter this mode you will see this menu:

The first slider affects the size of map (in tiles).

The second window affects the ground type. For more information look into this section.

The slider at the bottom modifies the default altitude level (land and water).

Two checkboxes on the right side taggle tile/altitude randomization on the created map.

When all settings are made, press OK to create a map or Cancel to abort.
