Keyboard Shortcuts
 
F1 Call this help manual
<CTRL> + N  Create New Map
<CTRL> + O  Open the map
<CTRL> + S  Save your map
<CTRL> + A Save as new map
<CTRL> + Z Undo/Redo the last operation
<CTRL> + B Select area for copy/paste
<CTRL> + X Cut selected area
<CTRL> + C Copy selected area
<CTRL> + V Paste selected area
<CTRL> + F4 Close the window
<SHIFT> + F1 Help about main window
<CTRL> + <SHIFT> + M  Turn on/off mini map
<CTRL> + 1  Zoom the map out 1:2
<CTRL> + 2  Zoom to normal view
<CTRL> + 3 Zoom the map in 2:1
<CTRL> + L  Turn on the light setting
<CTRL> + R  Turn on/off randomizing
<CTRL> + E  Randomizing of the entire map
<CTRL> + <SHIFT> + R  Randomizing altitude level throughout the map
<CTRL> + <SHIFT> + A  Turn on/off smoothing
<CTRL> + <SHIFT> + L Flat surface under buildings
<CTRL> + <SHIFT> + I Turn on/off  group number display
<CTRL> + <SHIFT> + T  Toggle time of the day changes
<CTRL> + <SHIFT> + V Show Units view ranges
<CTRL> + <SHIFT> + G Change global description and settings
<CTRL> + G Turn on/off the Grid
<Page Up>  Scroll up the map by 10 points
<Page Down> Scroll up the map by 10 points
<Space>  Edit terrain
<SHIFT> + 0..9 Fast terrain selecting  (terrain mode only)
<SHIFT> + R Switch to the fast square transition mode
<SHIFT> + A Switch to the altitude-editing mode
<SHIFT> + T Switch to the special square editing mode
<SHIFT> + O Edit objects
<SHIFT> + S Switch to edit buildings mode
<SHIFT> + B Switch to dymanic bridges mode
<SHIFT> + U  Switch to editing units mode
<SHIFT> + I Switch to items mode
<SHIFT> + G Switch to logic mode
<SHIFT> + M Switch to music mode
<CTRL> + M Define default music
<SHIFT> + L Searching for units by using unit ID
